Grid

Grid VS Flexbox

2D VS 1D

Will use both

Grid is NOT replacing Flexbox

Flexbox vs Grid

Understanding it requires its own talk

Quick overview

  1. Dividing the element into custom 2D grid
  2. Then positioning sub elements on that grid
  3. With full control of sizes, gaps and placements
  4. Fully responsive
  5. Children can overlap
  6. Think of lines, not just rows and columns

.wrapper {
	display: grid;
	grid-template-columns: repeat(3, 1fr);
	grid-gap: 10px;
	grid-auto-rows: minmax(100px, auto);
}

.child-Two {
	grid-column: 2 / 4;
	grid-row: 1 / 3;
}
